项目经历

CareerPilot 实习求职 Agent 工作流平台

2026.07 - 至今

技术栈：Python + FastAPI + React + TypeScript + Pydantic v2 + DeepSeek API + OpenAI-compatible API + ReportLab + Playwright(optional) + Docker Compose + Run Trace

项目面向学生实习投递场景，构建从岗位收集、简历/JD 结构化解析、匹配评分、能力缺口分析、 证据锁定改写到人工审批导出的 AI Agent 流程平台，强调可追踪、可复盘、可审计，避免生成虚假经历。

  • Agent 工作流与可观测性：基于 FastAPI 实现 LoopEngine 的 plan / execute / verify / reflect 流程，记录 run、step、event、checkpoint 和 LLM token/cost，支持人工审批、失败恢复和前端运行轨迹可视化。
  • 结构化解析与岗位收集：接入 DeepSeek API 输出 Pydantic 结构化简历/JD， 内置 JSON 修复与本地 fallback；W6 增加 JobCollectorAgent，支持用户提供公开 URL / HTML / 文本，进行正文抽取、公开页面安全校验、正文 hash、HTML hash 和可选截图留证。
  • 证据锁定匹配：实现 MatchAgent 对简历画像与 JD 画像进行要求匹配、关键词对齐、 证据映射和缺口分级，输出整体匹配分、缺失项、改写优先级，确保每条建议都能回溯到原始经历或岗位文本。
  • 简历改写与 HITL 审批：实现 ResumeRewriteAgent，把匹配结果转成中文改写草稿、 Diff、风险提示和证据链；没有证据的内容仅进入风险清单，不写入正式经历，审批通过后使用 ReportLab 导出支持中文字体的 PDF。
  • 产品化前端体验：使用 React + TypeScript 重构为面向用户的求职流程首页， 隐藏底层开发细节到开发者视图；实现玻璃拟态面板、滚动渐显、预加载、音乐联动粒子背景和模型处理动效， 使 Agent 的执行状态、成本、证据和审批过程直观可见。
  • 质量评测与成本可观测：实现 EvalHarness 和 QualityGate，对解析覆盖、 匹配证据、改写真实性、面试准备完整度和 CRM 记录质量进行评分；支持可选 LLM-as-Judge， 输出 PASS/WARN/BLOCK、HTML 报告，并记录 Judge token、延迟和成本。
  • 生产化收尾与安全边界：补充 Docker Compose、API Token 鉴权、限流、 安全响应头、readiness 检查和成本汇总接口；浏览器收集工具默认不携带 Cookie/登录态， 不访问 localhost/私有网段，不绕过验证码或平台限制，适合演示 Agent 的工具边界、 审计链路和人类审批设计。
